Functions and Scope

What are Functions?

Functions are reusable blocks of code that perform specific tasks. They help organize code and avoid repetition.

Function Types

  • Arrow Functions: const myFunction = () => {}

Understanding Scope

Scope determines where variables can be accessed in your code:

  • Global Scope: Variables accessible everywhere
  • Function Scope: Variables accessible only within the function
  • Block Scope: Variables accessible only within the block (let/const)

Best Practices

  1. Use descriptive function names
  2. Keep functions small and focused
  3. Use parameters instead of global variables
  4. Return values explicitly
